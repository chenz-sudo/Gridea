{"posts":[{"title":"NETCDF4&PyFVCOM的安装","content":" PyFVCOM is a collection of various tools and utilities which can be used to extract, analyse and plot input and output files from FVCOM as well as generate model inputs. Github：https://github.com/pwcazenave/PyFVCOM Step1 安装openmpi 下载软件包 https://download.open-mpi.org/release/open-mpi/v3.1/openmpi-3.1.4.tar.gz 解压 tar -xzvf openmpi-3.1.4.tar.gz 安装 cd openmpi-3.1.4 ./configure --prefix=/path to openmpi/ make make install 设置环境变量 vi ~/.bashrc 在文本末尾添加如下内容 export OMP=/path to openmpi/ export PATH=$OMP/bin:$PATH export LD_LIBRARY_PATH=$OMP/lib:$LD_LIBRARY_PATH export INCLUDE=$OMP/include:$INCLUDE 保存后更新环境 source ~/.bashrc Step 2 安装 hdf5 下载软件包 https://support.hdfgroup.org/ftp/HDF5/releases/hdf5-1.10/hdf5-1.10.5/src/hdf5-1.10.5.tar.gz 解压 tar -xzvf hdf5-1.10.5.tar.gz 安装 cd hdf5-1.10.5 ./configure --prefix=/path to hdf5/ --enable-fortran --enable-parallel CC=mpicc FC=mpif90 CXX=mpicxx make make install 设置环境 vi ~/.bashrc 在文本末尾添加如下内容 export HDF5=/path to hdf5/ export PATH=$HDF5/bin:$PATH export LD_LIBRARY_PATH=$HDF5/lib:$LD_LIBRARY_PATH export INCLUDE=$HDF5/include:$INCLUDE 保存后更新环境 source ~/.bashrc Step 3 安装 NETCDF-C 下载软件包 https://www.unidata.ucar.edu/software/netcdf/ 解压 tar -zxvf netcdf-c-4.7.0.tar.gz 安装 cd netcdf-c-4.7.0 ./configure --prefix=/path to netcdf/ LDFLAGS=-L$HDF5/lib CPPFLAGS=-I$HDF5/include CC=mpicc make make install 设置环境 vi ~/.bashrc 在文本末尾添加如下内容 export NETCDF=/path to netcdf/ export PATH=$NETCDF/bin:$PATH export LD_LIBRARY_PATH=$NETCDF/lib:$LD_LIBRARY_PATH export INCLUDE=$NETCDF/include:$INCLUDE 保存后更新环境 source ~/.bashrc Step 4 安装 NETCDF-Fortran 下载软件包 https://www.unidata.ucar.edu/software/netcdf/ 解压 tar -zxvf netcdf-fortran-4.4.5.tar.gz 安装 cd netcdf-fortran-4.4.5 ./configure --prefix=/path to netcdf/ LDFLAGS="-L$HDF5/lib -L$NETCDF/lib" CPPFLAGS="-I$HDF5/include -I$NETCDF/include" CC=mpicc FC=mpif90 F77=mpif90 make make install 更新环境 source ~/.bashrc 验证安装 nc-config --all 如果对编译NETCDF时的编译器特殊要求（如intel），在CC、FC等指定。 Step 5 安装PyFVCOM pip install PyFVCOM 过程中提示缺少组件另行安装即可。 或下载安装包安装 python setup.py build python setup.py install ","link":"https://chenz-sudo.github.io/Gridea/post/pyfvcom-install/"}]}
